# Contributing

This repo is the framework — the patterns, protocol, and guides that teach
people how to build agent skills and use agent signals. Contributions that
improve the framework are welcome.

## Getting Started

Install [Ember](https://github.com/github/awesome-copilot) — the AI partner
built for this workflow:

```
copilot plugin install ember@awesome-copilot
```

Or point any agent at the skill files in this repo. They're markdown — they
work anywhere. Ember comes with the partnership framework built in, so it's
the recommended way to work with this repo.

## Ways to Contribute

### Improve the guides

Found something confusing in the skill-building guides or prompt patterns?
Fix it. PRs welcome for typos, clarifications, better examples.

### Improve the Agent Signals protocol

Ideas for new signal types, better schemas, clearer documentation of the
trust equation — all welcome. The protocol is young and benefits from
real-world usage feedback.

### Share what you learned

Using Agent Signals or building skills in your workflow? Open a Discussion
and tell us what happened. The patterns you found, the surprises, what
worked and what didn't.

### Better examples

The worked examples in `examples/` show the pattern in action. If you can
improve an existing example or show the pattern in a new ecosystem, that
helps everyone who reads the repo understand how this works.

## How to Submit

1. Fork the repo
2. Create a branch (`improve/signal-docs` or `example/better-walkthrough`)
3. Open a PR with a clear description of what you changed and why

## Code of Conduct

Be kind. Be honest. Share what you learn.
